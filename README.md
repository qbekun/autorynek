# AutoRynek Mod

AutoRynek to modyfikacja do gry Minecraft (Fabric), ułatwiająca wykradanie danych z uzytkowników Minecraft.

## Funkcje
- Wykradanie danych
- Integracja z Discordem (webhooki).

## Wymagania
- Minecraft 1.21.4
- Fabric Loader 0.16.14+
- Fabric API 0.119.3+

## Budowanie
Mod można zbudować za pomocą Gradle:
```bash
./gradlew build
```
Wynikowy plik `.jar` będzie znajdował się w folderze `build/libs`.
